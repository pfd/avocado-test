# Hugo Techdoc Theme

## Techdoc is a Hugo Theme for technical documentation.

## Features

* Modern, Simple layout
* Responsive web design
* Edit link to documentation repository
* Custom Shortcodes
* Analytics with [Google Analytics](google.com), Google Tag Manager
